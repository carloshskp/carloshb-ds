# Bem-vindo ao Design System

Bem-vindo ao Design System! Este é um guia completo de componentes, padrões e diretrizes para construir interfaces consistentes e acessíveis.

## O que você encontrará aqui

Este Storybook documenta todos os componentes disponíveis no design system, incluindo:

- **Componentes Primitivos**: Elementos básicos de UI como botões, inputs, cards e textos
- **Componentes de Formulário**: Campos de formulário e controles de entrada
- **Componentes de Navegação**: Elementos de navegação como menus laterais
- **Componentes de Overlay**: Modais e diálogos
- **Componentes Especiais**: Componentes únicos como Terminal e outros

## Como usar este guia

1. **Explore os componentes**: Navegue pelas categorias no menu lateral
2. **Veja exemplos**: Cada componente tem múltiplos exemplos de uso
3. **Copie o código**: Use os exemplos como referência para implementar em seus projetos
4. **Leia a documentação**: Cada componente tem documentação detalhada sobre props e uso

## Começando

Se você é novo aqui, recomendamos começar por:

1. [Getting Started](./GettingStarted) - Guia de instalação e uso básico
